### About

This repo is made up of *parts* of different public wallpaper repos:

- [Gruvbox Wallpapers](https://github.com/AngelJumbo/gruvbox-wallpapers)
- [Walls](https://github.com/dharmx/walls)

Some were also converted to specific themes using [gowall](https://github.com/Achno/gowall)

Credits to the various artists
